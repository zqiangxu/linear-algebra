# 向量 

## 1、向量的加法

```math
\begin{pmatrix}
x_1 \\
\vdots \\
x_n
\end{pmatrix}

+

\begin{pmatrix}
y_1 \\
\vdots \\
y_n
\end{pmatrix}

=

\begin{pmatrix}
x_1 + y_1 \\
\vdots \\
x_n + y_n
\end{pmatrix}

```


## 2、向量的乘法

```math
c
\begin{pmatrix}
y_1 \\
\vdots \\
y_n
\end{pmatrix}
=
\begin{pmatrix}
cx_1 \\
\vdots \\
cx_n
\end{pmatrix}
```
