# 矩阵的加减法

## 1、矩阵加法

```math
\begin{pmatrix}
x_1 \\
\vdots \\
x_n
\end{pmatrix}

+

\begin{pmatrix}
y_1 \\
\vdots \\
y_n
\end{pmatrix}

=

\begin{pmatrix}
x_1 + y_1 \\
\vdots \\
x_n + y_n
\end{pmatrix}

```


## 2、矩阵减法

```math
\begin{pmatrix}
x_1 \\
\vdots \\
x_n
\end{pmatrix}

-

\begin{pmatrix}
y_1 \\
\vdots \\
y_n
\end{pmatrix}

=

\begin{pmatrix}
x_1 - y_1 \\
\vdots \\
x_n - y_n
\end{pmatrix}

```
